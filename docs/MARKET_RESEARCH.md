## 一、各市场交易规则

### 1.1 美股 (US Stock Market)

| 项目 | 详情 |
|------|------|
| 交易时间 | 9:30 - 16:00 EST (周一至周五) |
| 盘前交易 | 4:00 - 9:30 EST |
| 盘后交易 | 16:00 - 20:00 EST |
| 结算周期 | T+0 (当日结算) |
| 涨跌幅 | 无涨跌幅限制 |
| 最小交易单位 | 1股 |
| 交易税费 | 0.008% SEC Fee + 交易所费用 |
| 主要交易所 | NYSE, NASDAQ |

### 1.2 加密货币 (Cryptocurrency)

| 项目 | 详情 |
|------|------|
| 交易时间 | 7x24 小时全年无休 |
| 结算周期 | 即时结算 (T+0) |
| 涨跌幅 | 无限制 (高波动性) |
| 最小交易单位 | 0.0001 BTC 等 |
| 交易税费 | Maker/Taker 手续费 (0.1%-0.5%) |
| 主要交易所 | Binance, Coinbase, Kraken |
| 特点 | 高波动性、杠杆交易、合约市场 |

### 1.3 港股 (HK Stock Market)

| 项目 | 详情 |
|------|------|
| 交易时间 | 9:30 - 16:00 HKT (周一至周五) |
| 午间休息 | 12:00 - 13:00 |
| 结算周期 | T+2 |
| 涨跌幅 | 无涨跌幅限制 |
| 最小交易单位 | 1手 (不同股票不同) |
| 交易税费 | 0.2% 印花税 + 0.005% 结算费 |
| 主要交易所 | HKEX (港交所) |
| 特殊产品 | 涡轮 (Warrant)、牛熊证、窝轮 |

### 1.4 A股 (China A-Share)

| 项目 | 详情 |
|------|------|
| 交易时间 | 9:30 - 15:00 CST (周一至周五) |
| 午间休息 | 11:30 - 13:00 |
| 结算周期 | T+1 |
| 涨跌幅 | 10% (主板), 20% (创业板/科创板) |
| 最小交易单位 | 100股 (1手) |
| 交易税费 | 0.1% 印花税 + 0.002% 过户费 |
| 主要交易所 | 上交所 (SSE), 深交所 (SZSE) |
| 特殊规则 | 涨跌停板、T+1 限制、涨停板拆单 |

## 二、各市场股票代码格式

### 2.1 代码格式汇总

| 市场 | 示例 | 格式说明 |
|------|------|----------|
| 美股 | AAPL, NVDA, MSFT | 纯字母，1-5位 |
| 港股 | 00700.HK, 09988.HK | 数字 + .HK 后缀 |
| A股 (上海) | 600519.SS, 688111.SS | 6位数字 + .SS |
| A股 (深圳) | 000001.SZ, 300750.SZ | 6位数字 + .SZ |
| 加密货币 (USD) | BTC-USD, ETH-USD | 符号-USD |
| 加密货币 (USDT) | BTCUSDT, ETHUSDT | 符号USDT |

### 2.2 yfinance 支持格式

```python
# 美股 - 直接使用代码
stock = yf.Ticker("AAPL")

# 港股 - 需要 .HK 后缀
stock = yf.Ticker("00700.HK")

# A股 - 上海交易所使用 .SS
stock = yf.Ticker("600519.SS")

# A股 - 深圳交易所使用 .SZ
stock = yf.Ticker("000001.SZ")

# 加密货币 - 使用 USD 后缀
stock = yf.Ticker("BTC-USD")
```

## 五、注意事项

### 5.1 数据延迟

- **yfinance**: 延迟 15-30 分钟
- **AkShare**: 延迟 15 分钟 (部分数据)
- **API 实时数据**: 需要付费

### 5.2 访问限制

- 免费 API 有频率限制
- 需要处理网络超时和重试
- 数据缓存策略很重要

### 5.3 数据质量问题

- yfinance 对港股/A股支持有限
- 部分历史数据可能缺失
- 需要数据清洗和标准化
